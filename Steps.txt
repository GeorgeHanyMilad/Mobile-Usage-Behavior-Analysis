الداتا بتاعتي عبارة عن فايل CSV واحد بس و جايبه من Kaggle و ده اللينك: https://www.kaggle.com/datasets/valakhorasani/mobile-device-usage-and-user-behavior-dataset

الداتا بتاعتي فيها 11 عمود و 700 صف

أسماء الأعمدة و شرحها قبل ما أعمل أي حاجة في الداتا:
User ID, Device Model, Operating System, App Usage Time (min/day), Screen On Time (hours/day), Battery Drain (mAh/day), Number of Apps Installed, Data Usage (MB/day), Age, Gender, User Behavior Class 

- عمود الـ User ID معناه: Unique identifier for each user
- عمود الـ Device Model معناه: Model of the user's smartphone
- عمود الـ Operating System معناه: The OS of the device (iOS or Android)
- عمود الـ App Usage Time معناه: Daily time spent on mobile applications, measured in Minutes
- عمود الـ Screen on Time معناه: Average hours per day the screen is active
- عمود الـ Battery Drain معناه: Daily battery consumption in mAh
- عمود الـ Number of Apps Installed معناه: Total apps available on the device
- عمود الـ Data Usage معناه: Daily mobile data consumption in megabytes
- عمود الـ Age معناه: Age of the user
- عمود الـ Gender معناه: Gender of the user (Male or Female)
- عمود الـ User Behavior Class معناه: Classification of user behavior based on usage patterns (1 to 5)

الأسباب المحتملة (Possible Reasons)
- الفئة (36-45) يمكن يكونوا مشغولين أكتر بظروف العمل والأسرة فده قلل استخدامهم للموبايل و للتطبيقات و لكل حاجة.
- الفئة (18–25): عندهم فضول وتجربة مستمرة للتطبيقات الجديدة (Social Media, Games, Entertainment Apps) ووقت فراغ أكبر مقارنةً بالفئات الأكبر وبالتالي معدلات استخدامهم وتحميل التطبيقات أعلى.
- الفئة (46–59): ممكن يعتمدوا بشكل كبير على الموبايل كوسيلة أساسية للاتصال ومتابعة الأخبار والفيديوهات وده يفسر ليه استهلاكهم للبيانات وساعات الشاشة عالي جدًا.
- تقارب الذكور والإناث في App Usage: يرجع لانتشار واسع للتطبيقات الاجتماعية (WhatsApp, Facebook, Instagram, TikTok) اللي بيستخدمها الجنسين بنفس المعدل تقريبًا فمبقاش فيه فرق جوهري.
- العلاقة الطردية (App Usage ↔ Data Usage): منطقي لأن التطبيقات الأكثر استهلاكًا للوقت غالبًا معتمدة على الإنترنت (Social Media, Video Streaming).
- العلاقة الطردية (Apps Installed ↔ App Usage): المستخدم اللي عنده عدد تطبيقات أكبر بيكون عنده بدائل أكتر للترفيه أو العمل وبالتالي بينوع استخدامه وبيقضي وقت أطول.

الـ Recommendations (التوصيات):
- توعية الفئة 18–25 بأهمية إدارة الوقت على التطبيقات.
- تشجيع الفئة 46–59 على استخدام أدوات للتحكم في استهلاك البيانات.
- عمل حملة توعية عبر TikTok و Instagram و Facebook تستهدف الفئة 18–25 لزيادة الوعي بإدارة الوقت.
- تشجيع الفئة 26–35 على تقليل استخدام السوشيال ميديا وزيادة التطبيقات التعليمية/العملية.
- إضافة App Notification أو Reminder بعد 3 ساعات استخدام متواصل ينبه المستخدم لإدارة وقته.
